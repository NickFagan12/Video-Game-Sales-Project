# 🎮 Video Game Sales Data Analysis
Tableau Public Link: https://public.tableau.com/views/VideoGameSales_17531245474280/GlobalSalesbyGenre?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link


This project explores global video game sales using SQL for data analysis and Tableau for data visualization. It aims to uncover patterns and trends in the gaming industry from the 1980s to the 2010s, using the publicly available Video Game Sales dataset from Kaggle.

---

## 📊 Project Objectives

- Analyze sales trends across regions (NA, EU, JP, Others)
- Identify top-selling games, genres, platforms, and publishers
- Discover how video game popularity has evolved over time
- Create visual dashboards to communicate insights clearly

---

## 📁 Dataset

**Source:** [Kaggle - Video Game Sales](https://www.kaggle.com/datasets/gregorut/videogamesales)

**Columns:**
- `Name`: Game title
- `Platform`: Console/Platform (e.g., PS4, Xbox, PC)
- `Year`: Year of release
- `Genre`: Game genre
- `Publisher`: Publishing company
- `NA_Sales`, `EU_Sales`, `JP_Sales`, `Other_Sales`, `Global_Sales`: Sales figures in millions

---

## 🛠️ Tools Used

- **SQL Workbench**: Data cleaning, querying, and aggregation
- **Tableau Public**: Interactive dashboards and visual storytelling
- **Excel**: Minor preprocessing (optional)

---

## 📈 Key Visualizations (in Tableau)

- Global Sales by genre (Bar Graph)
- Global Sales by year (Line Graph)
- Regional Sales by platform (Bar Graph)
- Top 10 games by global sales (Bar Graph)
- Regional Sales by genre (Bar Graph)

---

## 🚀 How to Use

1. Clone the repo or download the project files
2. Use `video_game_sales.sql` to run analysis in SQL Workbench
3. Open the Tableau workbook (`.twb` or `.twbx`) to view and edit dashboards
4. Export visualizations as `.png` for reports or presentations

---

## 📌 Insights Gained

- Action and Sports genres dominate global sales
- North America consistently leads in video game purchases
- Nintendo and Electronic Arts are among the most successful publishers
- Sales peaked in the mid-to-late 2000s and began to decline afterward
